IMPORTANT: Create Your Own .env File
====================================

I've created the code that reads MongoDB connection from environment variables,
but YOU need to create the .env file with your actual credentials.

STEP 1: Create a file named ".env" (without quotes) in the root directory

STEP 2: Copy and paste this content into .env:

# ============================================
# AI-Powered Exam Suite - Environment Variables
# ============================================

# Server Configuration
PORT=5000
NODE_ENV=development

# MongoDB Configuration
# Choose ONE option below:

# Option 1: Local MongoDB
MONGODB_URI=mongodb://localhost:27017/ai-exam-suite

# Option 2: MongoDB Atlas (Cloud - replace with your actual connection string)
# MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/ai-exam-suite?retryWrites=true&w=majority

# Authentication
# Generate a secure random string (run in terminal):
# node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
JWT_SECRET=your-super-secret-jwt-key-change-this-in-production-min-32-chars
JWT_EXPIRE=7d

# OpenAI API Key (Optional - for backward compatibility)
# Get from: https://platform.openai.com/api-keys
OPENAI_API_KEY=sk-your-openai-api-key-here

# Groq API Key (Primary - for AI question generation)
# Get from: https://console.groq.com/keys
GROQ_API_KEY=your-groq-api-key-here

# Groq Model Selection (Optional)
# Options: 
#   - llama-3.1-8b-instant: Fast and free (recommended for most cases)
#   - llama-3.1-70b-versatile: More powerful (for complex questions)
# Default: llama-3.1-8b-instant (if not specified)
# GROQ_MODEL=llama-3.1-8b-instant

# File Upload Configuration
MAX_FILE_SIZE=10485760
UPLOAD_PATH=./uploads

# CORS Configuration
FRONTEND_URL=http://localhost:3000


STEP 3: Replace the placeholder values with your actual credentials

For MongoDB Setup:
- See MONGODB_SETUP.md for detailed instructions
- Local: mongodb://localhost:27017/ai-exam-suite
- Atlas: Get connection string from MongoDB Atlas dashboard

For OpenAI:
- Sign up at https://platform.openai.com
- Get API key from https://platform.openai.com/api-keys
- Note: You need a paid account for GPT-4

That's it! Now when you run "npm start", it will read this .env file and connect to MongoDB.
